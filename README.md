# AcidHarken

**Live site:** https://acidharken.netlify.app/

AcidHarken is a web-based amino acid composition calculator. Paste a protein sequence using single-letter amino acid codes, and it computes the percentage of each amino acid along with a weighted refractive index.

## Tech Stack

- React 19 + Vite 7
- MUI 7 (Material UI) with Emotion
- react-hook-form

## Development

```bash
npm install
npm run dev
```
